### Linux下解压命令

- filename.zip的解压

  ```
  unzip filename  .zip
  ```

- filename.tar.gz的解压

  ```
  tar -zxvf filename.tar.gz
  ```

其中zxvf含义分别如下

```
z:    	gzip				压缩格式
x:		extract			 解压
v:		verbose			详细信息
f:		file(file=archieve)		文件
```
